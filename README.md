# Capstone Project

This project is a containerized 3-tier web application.

## Technologies Used

- Flask (Backend API)
- Nginx (Frontend)
- Docker
- Docker Compose

## How to Run the Project

Run:

docker compose up --build

## Services

Frontend: http://localhost:8080

Backend API: http://localhost:5000

Backend Status Endpoint: http://localhost:5000/api/status
